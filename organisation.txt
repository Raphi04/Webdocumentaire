J'ai créer le fichier Jeremie.html mais c'est provisoire : si on code dans le même fichier on risque de faire des conflits GIT, en faisant ça on réduit les risques.
J'ai écrit une fonction Javascript qui reliera le code à index.html, quand on aura fini on copie-collera directement dedans et on supprimera Jeremie.html.
Pour le CSS, utilise secondContent.css (dans le dossier style), pareil j'ai relier les fichiers CSS secondaire au pricipale : style.css.
Pareil pour le JS, utilise Jeremie.js (comme pour le HTML, on copie-collera à la fin).

Donc :
- Code sur Jeremie.html
- Code sur secondContent.css
- Code sur Jeremie.js
